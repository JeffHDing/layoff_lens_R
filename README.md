# layoff_lens_R

Shiny R app for investigating hiring/layoff trends.

**Deployed application:** <https://YOUR_APP_URL.posit.cloud/>

## Install Packages

From an R console, install the required packages:

```r
install.packages(c("shiny", "bslib", "ggplot2", "dplyr", "scales"))
```

## Run the Application

```r
shiny::runApp()
```
